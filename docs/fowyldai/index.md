## Key Capabilities

- **Sovereign Inference** — Models run entirely within your infrastructure on local GPU/CPU
- **Air-Gap Ready** — Full offline deployment with no external dependencies
- **Crown Engine** — FastAPI server with 115+ subsystems, intelligent model routing via Sovereign Brain
- **Council Protocol** — Multi-model deliberation for complex tasks
- **OpenAI-Compatible API** — Drop-in `/v1/chat/completions`, `/v1/embeddings`, `/v1/models` endpoints
- **Enterprise Security** — SOC2-aligned, RBAC, tenant isolation, GDPR/AI Act compliance endpoints
- **VS Code MCP Integration** — 20+ tools exposed as an MCP server for developer workflows
- **Multi-Model Routing** — Auto-routes requests to the best model (phi3, mistral, openhermes, llama, gemma)
- **`fowyldctl` CLI** — Customer admin CLI for setup, model management, security, diagnostics
