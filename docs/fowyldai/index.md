# FowyldAI — The Fowyld Codex

**Sovereign AI inference that your organization owns and controls.**

FowyldAI is a private, air-gappable AI engine built for enterprises that need AI without sending data to third parties. Deploy on-premise or in your own cloud — your models, your data, your rules.

## Key Capabilities

- **Sovereign Inference** — Models run entirely within your infrastructure
- **Air-Gap Ready** — Full offline deployment with no external dependencies
- **Crown Engine** — Intelligent routing across local models
- **Council Protocol** — Multi-agent coordination for complex tasks
- **Enterprise Security** — SOC2-aligned, RBAC, tenant isolation
- **VS Code Integration** — MCP server for developer workflows
